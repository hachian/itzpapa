# Product Overview

itzpapaは、ObsidianユーザーのためのAstroブログソリューションです。Obsidianで執筆したコンテンツを、そのままAstroで高速な静的サイトとして公開できます。

## Core Capabilities

- **Obsidian互換記法**: WikiLink（`[[ページ名]]`）、Callout、マークハイライト（`==テキスト==`）をネイティブサポート
- **高速静的サイト生成**: Astro v5によるビルド時レンダリングで爆速表示
- **階層的タグシステム**: タグによるコンテンツ整理とナビゲーション
- **SEO最適化**: canonical URLs、OpenGraph、サイトマップ、RSSフィード自動生成

## Target Use Cases

- **知識管理 → 情報発信**: Obsidianで蓄積したノートをそのままブログとして公開
- **技術ブログ運営**: Markdownベースの執筆環境で効率的なコンテンツ作成
- **ドキュメント公開**: WikiLink記法を活かした相互参照のあるドキュメントサイト

## Value Proposition

「Obsidianで書いて、Astroで公開」- 知識管理と情報発信をシームレスに接続。
Obsidianの強力な記法（WikiLink、Callout、ハイライト）をそのまま使えるため、執筆環境を変えることなくブログを運営できます。
