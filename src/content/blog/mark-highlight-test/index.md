# マークハイライト機能のテスト

このページでは、マークハイライト機能（`==text==`記法）の動作を包括的にテストします。

## 1. 基本的なハイライト

### 1.1 単一ハイライト
基本的なハイライトの例：==重要な情報==

### 1.2 複数ハイライト
複数のハイライトを含む文：==最初のハイライト==と==二番目のハイライト==があります。

### 1.3 日本語テキスト
日本語のハイライト：==重要な日本語テキスト==

### 1.4 英語テキスト
英語のハイライト：==Important English Text==

## 2. インライン記法との組み合わせ

### 2.1 太字との組み合わせ

### 2.2 イタリックとの組み合わせ
イタリックとハイライト：==*重要なイタリックテキスト*==

### 2.3 コードとの組み合わせ
コードとハイライト：==`function highlightCode()`==

### 2.4 リンクとの組み合わせ
リンクとハイライト：==[重要なリンク](https://example.com)==

### 2.5 複合記法
複合記法のテスト：==***[重要な複合リンク](https://example.com)***==

## 3. 特殊文字とエスケープ

## 4. 空白とフォーマット

### 4.1 前後の空白

### 7.1 箇条書き

- 通常の項目と==ハイライト項目==の混在

1. ==ステップ1==: 最初の重要なステップ
2. ==ステップ2==: 次の重要なステップ
3. 通常のステップと==重要なポイント==
