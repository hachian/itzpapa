# マークハイライト機能テスト

このページでは、`==text==` 記法を使用したマークハイライト機能をテストします。

## 動作している機能

### 基本的なハイライト
==重要なテキスト== をハイライト表示できます。

### 複数のハイライト
==最初のハイライト== と ==二番目のハイライト== と ==三番目のハイライト== を同じ行に配置できます。

### 日本語サポート
==重要な日本語テキスト== も正常にハイライトされます。

### 空白の処理
== 空白を含むテキスト == も自動的にトリムされてハイライトされます。

### 特殊文字と数字
==価格: $29.99== や ==100% 満足保証== のような特殊文字も処理できます。

### 長い文章での使用
これは ==長い文章の中で重要な部分== をハイライトし、さらに ==別の重要な部分== も強調できることを示すサンプルです。最後に ==最終的なポイント== を明示します。

## 現在の制限事項

以下の機能は技術的制約により未対応です：

### インライン記法との組み合わせ（未対応）
- `==**太字ハイライト**==` - 太字とハイライトの組み合わせ
- `==*イタリックハイライト*==` - イタリックとハイライトの組み合わせ  
- `==[リンクハイライト](https://example.com)==` - リンクとハイライトの組み合わせ

### 単独のハイライト（部分的に未対応）
- `==単独テスト==` - 状況により動作しない場合があります

## スタイリング確認

### アクセシビリティ機能
- **高コントラストモード**: ブラウザの高コントラストモード設定で境界線が追加されます
- **ダークモード**: システムのダークモード設定で色合いが調整されます
- **アニメーション削減**: motion設定でアニメーションが無効化されます

### レスポンシブ対応
このページは様々な画面サイズで確認してください：
- デスクトップ表示
- タブレット表示  
- モバイル表示

### プリント対応
このページを印刷プレビューで確認すると、==ハイライト部分== が印刷に適した色合いで表示されます。

### 通常のマークダウン記法との組み合わせ
- **太字** と ==ハイライト== の混在
- *イタリック* と ==ハイライト== の混在
- `コード` と ==ハイライト== の混在
- [通常のリンク](https://example.com) と ==ハイライト== の混在
