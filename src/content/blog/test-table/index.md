## 長いコンテンツを含むテーブル

| 機能 | 説明 |
|------|------|
| ダークモード対応 | システム設定に応じて自動的にダークモードに切り替わります。手動での切り替えも可能です。 |
| レスポンシブデザイン | 画面サイズに応じてテーブルが水平スクロール可能になり、モバイルでも快適に閲覧できます。 |
| ゼブラストライプ | 偶数行に薄い背景色が適用され、行の識別が容易になります。 |
| ホバーエフェクト | マウスを行に重ねると背景色が変わり、現在見ている行が分かりやすくなります。 |

- [ ] 角丸が適用されている
- [ ] ダークモードで正しく表示される
