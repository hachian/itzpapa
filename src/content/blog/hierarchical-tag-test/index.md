# 階層タグのテスト

この記事は階層タグのツリービュー機能をテストするために作成されました。

## 技術系の階層タグ

### Web開発
- フロントエンド技術: React, Vue, Angular
- バックエンド技術: Node.js, Python, Java

### モバイル開発
- iOS開発: Swift, Objective-C
- Android開発: Kotlin, Java

## プログラミング言語

### JavaScript エコシステム
- JavaScript基礎
- TypeScript型安全性

### Python開発
- データサイエンス
- ウェブ開発

この記事により、以下の階層構造が作成されます：

```
tech/
├── web/
│   ├── frontend
│   └── backend
└── mobile/
    ├── ios
    └── android

programming/
├── javascript
├── typescript
└── python
```
